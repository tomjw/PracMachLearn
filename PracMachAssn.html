Model Buildng

A random forest calssifier is now built using the selected variables.

RFmodel <- randomForest(classe ~ .,
      data=training.complete.cut, 
      ntree=50,
      mtry=4,
      importance=TRUE,
      na.action=na.roughfix,
      replace=FALSE)
